## Bits Manipulation

**Decimal Number converts to Binary Number**
> Convert 13 to Binary Number

```
1) 13 % 2 = 1;  13 / 2 = 6
2) 6 % 2 = 0 ;  6 / 2 = 3
3) 3 % 2 = 1 ;  3 / 2 = 1
4) 1 % 2 = 1 ;  1 / 2 = 0

result: 1101
verification: 1 * 2^3 + 1 * 2^2 + 0 + 1 * 2^1 = 8 + 4 + 1 = 13
```
